데이터 전처리 순서

1. 데이터 수집 
	- 질병관리청의 국민건강영양조사의 2005~2024년도까지 데이터 다운로드(확장자.sas7bdat)

2. 데이터 탐색 및 이해 
	- 수집한 데이터를 수치형데이터와 범주형 데이터로 분리하고 그 데이터에 맞게 검증 및 분석

3. 결측치 처리
	- 수집한 데이터의 크기가 많기 때문에 결측치는 전부 drop처리

4. 이상치 제거
	- 이상치 확인 공복혈당수치가 비정상적으로 높은 데이터 하나를 발견
		-> 다른 수치 확인 결과 비정상 데이터가아니라 초고 위험군 환자인것을 판단하고 그대로 진행

분석
- 각 컬럼들을 수치화데이터 범주형 데이터로 분리후 각 데이터 형태에 맞는 검정 실시
	-> 수치형 데이터는 t-test검정 실시
		-> 정규성 검정을 시행해야 하지만 국민건강영양조사 샘플수가 수만개이기 때문에 
			통계학의 중심극한정리에 따라서 모집단의 분포와 상관없이 표본 평균의 분포는 정규분포로 따르게 된다.
			따라서 정규성 가정에 위배되더라도 t-test적용

	-> 범주형 데이터는 카이제곱 검정 실시
		-> 단, 카이제곱 검정은 기대빈도가 5이상이여야 하기때문에 거기에 대한 검증 추가

분석 결론 - 진짜 데이터 사용 
		어느 수치가 어느정도되어야 위험군인가?
		하나만 위험해도 위험한가 여러개가 위험해야 위험한가?

 - 건강한 사람의 임의 데이터와 위험한 사람의 임의 데이터 뽑아서 모델 테스트 결과
	건강군에 해당하는 사람은 0.67%로 아주 낮은 발병확률을 보여주는 반면
	위험군에 해당하는 사람은 12.53%로 비교적 아주 높은 발병확률을 보여주고 있습니다.
	여기서 왜 12.53%밖에 안나오지 라고 생각하실 수있지만 국민건강영양조사 원본 데이터에서
	심근경색 환자의 실제 유병률은 고작 5.7% 내외였습니다. 그런 관점에서 볼때 위험군의 사람의
	12.53%는 일반인 평균 위험도(5.7%)보다 2배가 넘는 수치를 보여주고 있습니다.
