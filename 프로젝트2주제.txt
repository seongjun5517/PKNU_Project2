 - 데이터셋 : Liver Patient Dataset
 - 데이터셋 내용
	-> 나이 
	-> 성별
	-> TB-총 빌리루빈 : 적혈구가 파괴될 때 생기는 황색 색소입니다. 수치가 높으면 황달을 의심합니다.
	-> DB-직접 빌리루빈 : 간을 통과한 후의 빌리루빈입니다. 간이나 담관에 문제가 있을 때 수치가 상승합니다.
	-> alkphos-알칼리성 인산가수분해효소 : 뼈나 담관에 많이 존재하는 효소입니다. 담관 폐쇄나 간 질환 시 수치가 올라갑니다.
	-> sgpt (ALT)-알라닌 아미노전이효소 : 간 세포가 손상되었을 때 혈액으로 흘러나오는 가장 대표적인 효소입니다.
	-> sgot (AST)-아스파르테이트 아미노전이효소 : 간 외에 심장, 근육에도 존재하지만, 간 손상 시 ALT와 함께 수치가 상승합니다.
	-> TP-총 단백질 : 혈액 내 단백질의 총합입니다. 간의 합성 기능을 나타내는 지표입니다.
	-> ALB-알부민 : 간에서만 생성되는 단백질입니다. 간 기능이 떨어지면 수치가 감소합니다.
	-> a/g ratio-알부민/글로불린 비율 : 알부민과 글로불린의 비율입니다. 간경화나 신장 질환 예측에 중요한 지표입니다.
	-> target-결과 : 질환유무
